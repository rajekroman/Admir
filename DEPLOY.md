# Nasazení na GitHub Pages

1. Vytvoř nový veřejný GitHub repozitář.
2. Rozbal distribuční ZIP.
3. Nahraj všechny soubory a složky do kořene větve `main`.
4. Zkontroluj, že `index.html` leží přímo v kořeni.
5. Zachovej skrytou složku `.github/workflows`.
6. Otevři `Settings → Pages` a nastav zdroj `GitHub Actions`.
7. V kartě `Actions` počkej na zelený běh `Deploy GitHub Pages`.
